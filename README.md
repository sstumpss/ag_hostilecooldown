# AG HOSTILE COOLDOWN
A configurable hostile cooldown script for QBOX with whitelisted activities and jobs and commands for admins.

### Made for Astral Gaming.

## Requirements:
- **ox_lib** : https://github.com/overextended/ox_lib
- **Ace Permissions**


## Compatability:
Only tested and working with the following:

- **qbx_core** : https://github.com/Qbox-project/qbx_core
- **wasabi_ambulance** : https://forum.cfx.re/t/esx-qbcore-wasabis-advanced-ambulance-job/4914860
- **pug-paintball** : https://forum.cfx.re/t/fivem-pug-paintball-qbcore/4894094
- **nass_paintball** : https://forum.cfx.re/t/esx-qbcore-nass-advanced-paintball/5214355


## Normal Commands
- **hc_trigger {PlayerID}** : Forces a hostile cooldown trigger on a Player.
- **hc_removecooldown {PlayerID}** : Removes cooldown for a Player.
- **hc_togglecooldown {ON/OFF}** : Turns hostile cooldown on or off for whole server.

## Debug Commands
- **hc_testdie** : forces a test of the hostile cooldown on the client that executed it.
- **hc_teststart** : forces a start of the hostile cooldown on the client that executed it.
